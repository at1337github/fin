# Fin

## Overview
This project aims to sort out the relationship between the PayPal audit file and its source files, specifically the other two related files. The goal is to uncover additional data sources influencing the audit file, verify the accuracy of the entire audit, and ensure consistency.

## Tasks
- Analyze and determine the relationship between the PayPal audit file and its corresponding source files.
- Investigate if the audit file contains data from additional, undocumented sources.
- Verify the accuracy of the entire audit process.

## Future Goals
- Ensure accuracy by automating data validation tasks.
- Document step-by-step processes for future audits.
